# LMS Forntend/client

### Setup instructions

1. Clone the project

```
  git clone https://github.com/PrinceInScripts/LMS-Website/tree/main/client
```

2. Move into the directory

```
 cd client
```

3. Install the dependencies

 ```
 npm install
 ```

 4. Run the server
  
```
  npm run dev
```
